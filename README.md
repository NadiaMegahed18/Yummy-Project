# 🍲 Yummy - Food Recipes App

An interactive web application that helps users discover delicious food recipes from various categories using a Meal API.

## ✨ Features
* **Explore Meals:** Browse a wide variety of meals and cuisines.
* **Detailed Recipes:** View ingredients, measurements, and cooking instructions for each dish.
* **Search Integration:** Easily find specific meals by name.
* **Category Filtering:** Filter meals by area (e.g., Egyptian, Italian) or categories.

## 🛠️ Tech Stack
* **HTML5 & CSS3**
* **JavaScript** (AJAX / Fetch API)
* **jQuery** 
* **Bootstrap** (For responsive layout)
